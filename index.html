Discovery process 1 - Health trends

  • Observing the percieved health over the years, we notice some interesting patterns
  • PerHea + 1989-1993 - We start at a value 0,6 (or 60%)
  • PerHea + 1994-1999 - Stepping through the years, we see a rising trend. Are people getting healthier?
  • PerHea + 1999-2004 - Interestingly, there is a lot of missing in this wave. Did Y2K wipe out some data?
  • PerHea + 2005-2007 - Back on track! Numbers are still rising...
  • PerHea + 2010-2014 - Interesting! Comparing to the the previous wave, many people perceived happiness declines. Why is that?

NOTE: I am aware correlation does not imply causation. My observations should not be considered as fact

Discovery process 2 - What causes happiness?

  • Can money buy happiness? Let's investigate...
  • FinSat + 1999-2004 - We see here that Japan, Mexico, Nigeria and U.S.A had the population with the highest financial satisfaction
  • Happiness + 1999-2004 - Switching over to 'Happiness', we see that the same countries are topping the charts
  • Happiness + 1989-1993 - Going back to the 80's we see that Bulgaria was the least happy country
  • FinSat + 1989-1993 - Furthermore, we see that the Bulgarian population had the lowest financial satisfaction, further proof that money=happiness

NOTE: I am aware correlation does not imply causation. My observations should not be considered as fact

My learnings

I found this project to be very challenging, although I have learned a lot from it. The biggest thing I will take away from this project is that I should plan my visualization more carefully before I start making it. Using a bar chart felt like a good way of representing the data for the longest time until I had to use my visualization to do the discovery processes and now I wished I would have done things differently. I do think a bar chart could work if the visualization had more interactivity but it is quite hard to implement (for me).
